# PBLKmean
Beginner-friendly Machine Learning Project for Customer Segmentation

--> Behavioral Segmentation for Targeted Marketing
This project focuses on customer segmentation using machine learning techniques. The goal is to group customers based on their purchasing behavior to enable targeted marketing strategies.
The dataset is based on the Mall Customer Segmentation Dataset from Kaggle. Since the dataset does not directly provide Annual Spending and Purchase Frequency, these features were engineered using available attributes such as Annual Income and Spending Score.

--> Pipeline
Retail Data → Feature Engineering → StandardScaler → K-Means Clustering → Elbow Method → Customer Personas

--> Key Concepts Use
- Data Preprocessing
- Feature Engineering
- StandardScaler
- K-Means Clustering
- Elbow Method (Optimal K selection)
- Data Visualization

--> Objective
To identify different types of customers (personas) such as high-value customers, regular customers, and low-engagement customers, which can help businesses design personalized marketing strategies.

-->Output
- Clustered customer groups
- Visualization of clusters
- Business insights for targeted marketing
